### 코드

```c++
#include <iostream>
#include <string>

using namespace std;

int main(int argc, char const *argv[])
{
    int n;
    cin >> n;

    string result;
    int cnt = 0, num = 665;

    while (1)
    {
        string temp = to_string(num);
        if (temp.find("666") != -1)
        {
            cnt++;
        }
        if (cnt == n)
        {

            result = temp;
            break;
        }
        else
            ++num;
    }
    cout << result << endl;
    return 0;
}


```
<br/>
- 666부터 1씩 증가해 string::find함수로 666을 찾으면 cnt를 증가하고

- cnt와 입력값이 같으면 break하고 출력한다.
